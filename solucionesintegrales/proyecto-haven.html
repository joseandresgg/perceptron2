Proyecto HAVEN

Hadoop

Trabaja con volúmenes masivos de datos distribuidos. La estrategia abierta de HP combina las mejores soluciones y los mejores servicios Hadoop extremo a extremo.

Autonomy

Procesa e indexa todo tipo de información. Autonomy IDOL permite extraer valor de la información humana para obtener una ventaja competitiva.

Vertica

Analiza grandes volúmenes en tiempo real. Vertica proporciona una plataforma de análisis SQL nativo en tiempo real. Análisis de datos con muy baja latencia.

Enterprise Security

Recolección en tiempo real de eventos de seguridad. Solución universal de gestión de seguridad y logs. Recolecta, normaliza y categoriza eventos, logs, flujos de cualquier dispositivo en tiempo real.

n-Apps

Conectores, APIs y capacidades para construir n-Aplicaciones. Solución universal de gestión de seguridad y logs. Recolecta, normaliza y categoriza eventos, logs, flujos de cualquier dispositivo en tiempo real.
